# floating-memo.app

floating-memo app in node-webkit.

![img](http://monosnap.com/image/DS3rw7zNhQu3npC5yhBu6wjbIkLvaV.png)

## Feature

- Always on top
- Markdown editor(gfm support)
- Automatically saving previous writing memo.
- Save to DayOne
